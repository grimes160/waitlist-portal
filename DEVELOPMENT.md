# Development Notes

## AI Co-Pilot Workflow
This project uses an AI-first development strategy. Each GitHub Issue is written
as a strict functional specification designed for immediate handoff to an AI
assistant for code generation.

### How It Works
1. A task is identified and added to the GitHub Projects board
2. The issue is written as a functional spec with explicit acceptance criteria
3. The spec is handed off to an AI assistant for initial code generation
4. Output is reviewed, tested, and modified before merging
5. Deviations from AI output and the reasons for them are noted here
